關於Let's Talk 的關鍵字

開放政府

開放政府的四個要素為透明、參與、涵容、課責，期待達成公部門與公民社會彼此互信、共創合作。Let's Talk 的期待便是達到擴大參與、公私共創。如果想多知道開放政府是什麼，可以參考 開放政府聯絡人指南 了解在臺灣怎麼推動開放政府。

  • Read More

審議民主

審議式民主具有明確的目標，希望可以讓公民透過知情而理性的討論方式探討公共議題，過程中可以更加深入了解議題、知道其他公民的想法，並以一個公開協商的方式來取得共識。審議式民主的規範原則有：相互性原則、公開性原則、基本自由原則、基本機會原則和公平機會原則

  • Read More

心理健康

心理健康包含的內容多元，如：主觀幸福感、自覺效能、自主性、和其他人的互動、實現個人智能及情感潛力等內涵。無論學青或社青，都可能面對不同心理健康議題，也都涉及個體、環境與制度等各層面。疫情之下，也可檢視政策與社會可否提供足夠的因應與支持體系。

  • Read More

政策歷程

公共政策的形成階段包含了問題分析、政策規劃、政策合法化、政策執行以及政策評估。問題分析須釐清問題全貌，包含現象、近因與根因，是非常重要的基石。釐清問題後透過邀集不同利害關係人來確定政策方向與方案，理解建議可能會對哪些人造成哪些影響。

  • Read More

階段任務

辦一場 Talk 包含了啟動、籌備、執行、結案追蹤四個階段。
啟動與籌備旨在確定目標、分工以及技術演練、活動計畫；執行當日重點在如何控場以及紀錄；結案包含了整理成果、組織內檢討以及慶賀，宣告專案結束。在 Let's Talk 中，單場 Talk 結案仍有後續第二階段與成果會可參與。

想一想

想一想下列情境，思考你會怎麼做⋯⋯

你是一名警察，面對社區流行，你在受了訓練後，被派協助疫調。身為疫調者，這次任務的主要目標是找到個案可傳染期間的接觸者，匡列隔離。現在你已經拿到了個案的聯絡資料，要開始聯繫進行疫調。

請先開始下面的情境思考，再回頭點開查看我為什麼請您思考這個情境疫情調查也需使用積極聆聽的技巧，積極聆聽並不只是專心安靜地聽更包含怎麼適時提出問題、鼓勵、解釋以及反應感受與總結，這是在審議場合中很重要的技巧，對於主辦團隊來說，如何協助提供相關的環境並理解主持人的任務也是關鍵。

辦理一場審議活動跟疫調一樣，除了要聆聽與事實解答與詢問問題，有知情、足夠的資訊也至關重要；此外，也都有明確的目標，且並非導向一個預期的固定答案，同時建立信賴也是重要環節。因此先以靠近時事的疫調情境著手，讓大家之後可以回過頭來思考，以情境為借鏡，我們能怎麼運用一些相類的原則來辦一場審議活動。

在聯絡個案之前，你認為該做好哪些準備？

這是我的想法，提供參考在聯絡個案前，需要先做好準備，確定這次的訪談目的以及具備相關知識，以釐清要拋出的問題，此外也要思考有可能會被詢問什麼問題，先暸解相關資源與資訊。

你的前兩通電話，兩位確診者都不願意卸下心防，消極配合你只好請同事協助，你認為撇除可能存在的個案問題，你自己可以怎麼精進

這是我的想法，提供參考有些人在語意表達、語言能力、文化敏感度不足或是性格上在疫調時遇到一些問題，然而這些問題仍可通過行政協助或者加強練習來改進。雖然可能有些人原本就因為「聲音好聽溫柔」、「擅長聊天」等因素可能讓受訪者容易接受，但每個人仍能透過練習來發現符合自己風格又能成功完成任務的模式。練習可以先透過三人一組，一人扮演受訪者，一人擔任訪員，一人觀察並回饋，由觀察者與受訪者提供回饋。

個案接聽電話了，你認為在開始提問前，要先做哪些事？

這是我的想法，提供參考在剛開始的時候應該先自我介紹並且確定對方身分，建立信任也很重要，包含提供明確訊息，適時詢問對方有無疑問。比如在告知對方確診後，先提供「接下來要注意什麼」等明確訊息，也可適時詢問「還有沒有什麼問題」並且提供相關資訊與資源。當然，你也可以在這個階段理解個案比較喜歡被稱呼的方式，以拉近距降低防衛，讓個案覺得在接下來的訪談中會是舒適、自在的。

接著要進到訪談環節了，現在是社區流行期間，你的首要目標是要知道在個案可傳染期期間曾經接觸到哪些人，並且即時匡列、隔離。已知可傳染期為發病前三天到住院隔離前，你認為在這樣的情況下，要怎麼拋出問句已取得你想得知的訊息？

這是我的想法，提供參考開放式的問句通常能獲得比較多的資訊，勁量避免使用「是否」類型的問句。詢問「跟我說說你的工作在做些什麼吧」而不是「你這週有沒有跟別人一起工作」。由此也會注意到，由於我們已經事先知道需要取得哪些資訊，雖然並未有預期的固定解答，但仍可由此先準備一些好用的開放用句。

你在詢問的過程中，對方丟出了一些錯誤訊息，對疾病的認知有些錯誤，也影響了告知活動史的意願，這時你會怎麼做

這是我的想法，提供參考提供正確的衛教知識是進行疫情調查時也同樣重要的事情，正確的資訊與指引也能協助確診者

訪問過程中，因為你營造了一個良好的信賴環境，因此對方終於坦承這陣子常常跟朋友去 KTV 喝酒唱歌，隨後他又回到其他敘述上，並未多談。你知道，這是一個高風險情境，因此想要瞭解更多，這時你會怎麼詢問對方？

這是我的想法，提供參考一樣，開放性問句是一個更適合的詢問方式，但此時的提問有另一個任務就是聚焦問題，在這裡，我們想要了解的恐怕不是他唱了什麼歌，因此要詢問的重點會是他在哪些時間點去哪些KTV跟誰一起唱歌，然而過程中很有可能對方不記得細節，我們可以從不同的問句以及資訊協助詢問，比方請對方查看他的簡訊、Line跟行事曆，協助記起去了哪裡；請對方查看社群媒體與手機相簿，協助記起有哪些人一起去。

訪問很順利，你做了許多的筆記，此時你會怎麼樣與對方確認你的理解沒有錯誤

這是我的想法，提供參考除了過程中能夠適時地簡短重述，這也是展現有專心聆聽，增加對方分享意願的技巧，同時也能夠確認你的認知與對方是否相同。在最可以簡短的總結對方的描述，並讓對方瞭解你理解了他的想法。倘若最後對方有些正向的回饋，你也能給予鼓勵

這次的成功讓你很有信心，之後你又需要聯繫另一名個案。這次訪談的過程中，對方聊得很開心，聊了很多，對於你而言，你認為什麼是自己需要注意的

這是我的想法，提供參考對方願意放下心防是好事，可是我們仍不能忘記一定要進行的部分，也就是這次的主要目標千萬要記得自己的目標是什麼，並且以對應的問句來詢問需要獲得的資訊。

在另外一場訪談中，個案因前面才接到衛生局的電話，情緒很差、很激動，這時你該怎麼做

這是我的想法，提供參考訪員給予同理心回應是重要的，在回應時可以肯認對方的情緒反應，比如「公部門這麼多的訊息跟介入，讓你感覺很辛苦、不太舒適嗎？」你能先簡單處理情緒，詢問對方感受，之後給予相關的資訊與資源，如果對方狀態不好或在不適合的環境，也可另外約時間。

情境模擬結束，你認為疫調的情境與審議活動有哪些類似的地方可以參考？差別又在哪呢？

參與角色

這邊主要紀錄辦理過程中可能接觸到的人

  • 審議主持人
    • 大場主持人：說明與統整各組結論、總結
    • 分組主持人：於各小組引導與歸納
    • 紀錄：審議文字紀錄可以不由審議主持人訓練者擔任，但一般來說若有相關訓練者會較容易理解流程並協助歸納
  • 工作人員：非主辦團隊籌備人員但活動當日會來協助
  • 執行單位 Cheers：過程與結案的行政事項聯繫，或許也會有訪談或額外紀錄
  • 主辦機關 青年署：相關規定的制定與認定
  • 審議業師：協助團隊最佳化審議流程與議題
  • 台經院：專案政策幕僚單位，協助 talk 後續議題盤整
  • 觀察員：基於對整體專案評估目的參與各場 talk 進行觀察，未來提供青年署作為專案策進。可能也會有訪談環節。
  • 政府出席者：部分場次會有中央或地方的相關業務承辦人等出席活動了解大家的討論意見
  • 參與者：報名參加活動者

待辦清單

這邊主要紀錄共同待辦事項，各團隊的細項仍請照實際討論的狀況為準

整體而言，整個活動分成四大階段：啟動、籌備、執行、結案追蹤。

啟動的目的是要為接下來的正式籌備做準備，而籌備則是就議題、審議技術與規劃、活動場地/技術、活動流程、人員邀約與訓練、行政核銷、疫情因應措施等面向上進行規劃與處理。

執行階段是指活動當天，儘管有了先前的準備，現場仍要有監測機制，此外核對審議記錄以及進行整體活動記錄也是當天重要任務。

到了結案追蹤階段，多指第一階段的 Let's Talk 已經結束，此時需要進行行政核銷以及結案報告、成果紀錄的製作，除此之外，政策面上追蹤第二階段 Let's Talk 以及成果交流會的相關訊息並盡量出席，在審議討論中屬於個人或組織行動的部分，也可規劃揪團策進。

比較要注意的是在各階段內的待辦任務不一定有明確的先後順序，可能需同時處理。

建議這份清單的使用方式為：團隊根據所列項目，協調好內部的分工與死線，並登記在群組或行事曆等團隊成員均能收到且方便查取的平台。

若衡量有必要，也可以讓小工知道，小工可以協助提醒。
