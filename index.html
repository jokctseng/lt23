關於Let's Talk 的關鍵字

開放政府

開放政府的四個要素為透明、參與、涵容、課責，期待達成公部門與公民社會彼此互信、共創合作。Let's Talk 的期待便是達到擴大參與、公私共創。如果想多知道開放政府是什麼，可以參考 開放政府聯絡人指南 了解在臺灣怎麼推動開放政府。

  • Read More

審議民主

審議式民主具有明確的目標，希望可以讓公民透過知情而理性的討論方式探討公共議題，過程中可以更加深入了解議題、知道其他公民的想法，並以一個公開協商的方式來取得共識。審議式民主的規範原則有：相互性原則、公開性原則、基本自由原則、基本機會原則和公平機會原則

  • Read More

淨零碳排

淨零碳排包含的面向多元，如：公正轉型、能源轉型、綠色生活等等。不同的產業與生活中，都可能可以用不同的角度來面對及探討這個議題帶來的衝擊或所需的準備。

  • Read More

政策歷程

公共政策的形成階段包含了問題分析、政策規劃、政策合法化、政策執行以及政策評估。問題分析須釐清問題全貌，包含現象、近因與根因，是非常重要的基石。釐清問題後透過邀集不同利害關係人來確定政策方向與方案，理解建議可能會對哪些人造成哪些影響。

  • Read More

階段任務

辦一場 Talk 包含了啟動、籌備、執行、結案追蹤四個階段。
啟動與籌備旨在確定目標、分工以及技術演練、活動計畫；執行當日重點在如何控場以及紀錄；結案包含了整理成果、組織內檢討以及慶賀，宣告專案結束。在 Let's Talk 中，單場 Talk 結案仍有後續第二階段與成果會可參與。

參與角色

在一場 Talk 中團隊可能接觸哪些人？除了審議業師以及參與者外，還有審議主持人、Let's Talk 執行團隊 Cheers 雜誌、觀察員、台經院團隊。釐清不同角色與團隊的關係對於後續溝通以及執行都會有幫助喔。

  • Read More

參與角色

這邊主要紀錄辦理過程中可能接觸到的人

  • 審議主持人
    • 大場主持人：說明與統整各組結論、總結
    • 分組主持人：於各小組引導與歸納
    • 紀錄：審議文字紀錄可以不由審議主持人訓練者擔任，但一般來說若有相關訓練者會較容易理解流程並協助歸納
  • 工作人員：非主辦團隊籌備人員但活動當日會來協助
  • 執行單位 Cheers：過程與結案的行政事項聯繫，或許也會有訪談或額外紀錄
  • 主辦機關 青年署：相關規定的制定與認定
  • 審議業師：協助團隊最佳化審議流程與議題
  • 台經院：專案政策幕僚單位，協助 talk 後續議題盤整
  • 觀察員：基於對整體專案評估目的參與各場 talk 進行觀察，未來提供青年署作為專案策進。可能也會有訪談環節。
  • 政府出席者：部分場次會有中央或地方的相關業務承辦人等出席活動了解大家的討論意見
  • 參與者：報名參加活動者

待辦清單

這邊主要紀錄共同待辦事項，各團隊的細項仍請照實際討論的狀況為準

整體而言，整個活動分成四大階段：啟動、籌備、執行、結案追蹤。

啟動的目的是要為接下來的正式籌備做準備，而籌備則是就議題、審議技術與規劃、活動場地/技術、活動流程、人員邀約與訓練、行政核銷、疫情因應措施等面向上進行規劃與處理。

執行階段是指活動當天，儘管有了先前的準備，現場仍要有監測機制，此外核對審議記錄以及進行整體活動記錄也是當天重要任務。

到了結案追蹤階段，多指第一階段的 Let's Talk 已經結束，此時需要進行行政核銷以及結案報告、成果紀錄的製作，除此之外，政策面上追蹤第二階段 Let's Talk 以及成果交流會的相關訊息並盡量出席，在審議討論中屬於個人或組織行動的部分，也可規劃揪團策進。

比較要注意的是在各階段內的待辦任務不一定有明確的先後順序，可能需同時處理。

建議這份清單的使用方式為：團隊根據所列項目，協調好內部的分工與死線，並登記在群組或行事曆等團隊成員均能收到且方便查取的平台。

若衡量有必要，也可以讓小工知道，小工可以協助提醒。
